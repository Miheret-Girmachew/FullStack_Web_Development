The three methods of adding instructions to HTML documents,

Internal (adding CSS instructions on the HTML file/ on-file)

An internal CSS is defined in the "head" section of an HTML page, within a "style" element.

Inline

In inline, styling is directly applied inside the opening tag of the HTML element. For that, we use the "style" attribute of an HTML element.

External CSS file

Create a separate CSS file containing all the CSS instructions and to link this CSS file to the HTML file.
